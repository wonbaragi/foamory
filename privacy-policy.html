포머리 개인정보처리방침

포머리(이하 "앱")는 사용자의 개인정보를 소중하게 생각하며, 관련 법령을 준수하기 위해 다음과 같이 개인정보처리방침을 안내합니다.

앱 이름: 포머리 (Foamory)
패키지명: app.foamory.mobile

1. 수집하는 정보

현재 버전의 포머리는 회원가입, 로그인, 서버 전송 기능 없이 동작하며, 사용자가 입력한 정보는 기본적으로 사용자의 기기 내부에 저장됩니다.

항목 처리 방식 목적
세차 일지 정보 기기 내부 저장 작성한 세차 일지 조회, 수정, 이미지 내보내기 기능 제공
사용자가 추가한 케미컬/브랜드 정보 기기 내부 저장 자동완성 및 개인화된 입력 편의 제공
사진 저장 권한 사용자 동의 후 사용 세차 일지를 이미지로 저장하는 기능 제공

현재 버전의 앱은 사용자의 세차 일지, 메모, 사용자 추가 상품 정보를 개발자 서버로 전송하거나 외부와 공유하지 않습니다.

2. 개인정보의 이용 목적

  • 세차 일지 작성, 조회, 수정, 삭제 기능 제공
  • 사용자 추가 상품 및 브랜드 자동완성 기능 제공
  • 일지 이미지 저장 기능 제공
  • 앱 안정성 및 기본 기능 유지

3. 개인정보의 보관 및 처리 위치

앱에서 사용자가 입력한 데이터는 사용자의 기기 내부 저장소에 보관됩니다. 현재 버전에서는 별도의 회원 계정이나 원격 서버에 사용자 데이터를 저장하지 않습니다.

4. 개인정보의 제3자 제공

현재 버전의 포머리는 사용자의 개인정보를 제3자에게 제공하지 않습니다.

5. 권한 사용 안내

사진/저장 관련 권한

사용자가 세차 일지를 이미지 파일로 저장하려는 경우에만 사진 저장 관련 권한을 요청할 수 있습니다. 해당 권한은 이미지 저장 기능 제공을 위한 목적에 한해 사용됩니다.

6. 보관 기간 및 삭제

사용자가 앱에 입력한 데이터는 사용자가 직접 삭제하거나 앱을 삭제할 때까지 기기 내부에 보관됩니다.

  • 세차 일지 및 사용자 추가 상품 정보는 앱 내부 저장소에 유지됩니다.
  • 앱 삭제 시 기기 내부에 저장된 데이터는 운영체제 정책에 따라 함께 삭제될 수 있습니다.

7. 이용자의 권리

사용자는 언제든지 앱 내에서 입력한 데이터를 수정하거나 삭제할 수 있습니다. 앱 삭제를 통해 기기 내부 저장 데이터를 제거할 수도 있습니다.

8. 아동의 개인정보

포머리는 아동을 대상으로 개인정보를 수집하거나 별도로 처리하기 위한 서비스를 제공하지 않습니다.

9. 개인정보처리방침 변경

이 개인정보처리방침은 앱 기능 변경, 정책 변경, 법령 변경에 따라 수정될 수 있습니다. 내용이 변경될 경우 본 페이지의 시행일을 갱신하여 안내합니다.
